I.  Introducción

El programa FireForest.py es capaz de analizar una cantidad de imágenes dado un grid, analiza el RBG promedio, desviación estándar, Saturación y Brillo de cada Grid de una imagen. Utiliza funciones de la librería CV2 para análisis de las imágenes.

II.  Marco Teórico (reparación del Trabajo)

OpenCV [1](Open Source Computer Vision) es una librería open source de visión por computador, análisis de imagen y aprendizaje automático.

OpenCV es una librería multiplataforma disponible para Windows, Mac, Linux y Android distribuida bajo licencia BSD. Puede programarse con C, C++, Python, Java y Matlab.

III.  Documentacion

 

Constantes

GRID_COLUMS: Numero de columnas

GRID_ROWS: Numero de Filas

TOTAL_GRIDS: Numero totales de Gind dada la multiplicacion de columnas y filas

 

Funciones

1.      argumentos(): Lee los argumentos de la línea de comandos donde los parámetros son:

·        h, --help     retorna información de uso del programa

·        -f, --file     recibe un archivo de imágenes

·        --version      retorna la versión del programa

la función retorna una lista con tolas las líneas leídas.

 

2.      procesaArchivo(): Función que procesa las líneas leídas de un archivo para el uso del programa, retorna una arreglo de 2 elementos donde el 1ro es el nombre del archivo imagen y el segundo es la serie de atributos por grid (H=humo,N=no incendio,I=incendio)

3.      main():Esta función se encarga que ir archivo por archivo procesando la imagen en grids obteniendo el rgb promedio, desviación estándar, brillo y saturación y escribir en un archivo .arff para su análisis en weka.

 

IV.  Resultados

Se analizaron 235 imágenes y se generon una base de datos llamada ForestFire.arff para su estudio en weka.

V.  Conclusiones

Este programa permite analizar las diferentes métricas que se pueden analizar en una imagen para poder llevar a acabo un trabajo de minería y de entrenamiento supervisado, en este caso para detección de incendios forestales.
